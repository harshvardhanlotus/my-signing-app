डिजिटल साइन ॲप्लिकेशन - पहिल्यांदा सेटअप

Documents साइन करण्यासाठी, एक छोटं ॲप्लिकेशन तुमच्या कम्प्युटरवर install करायला लागेल. ही एक सुरक्षित आणि फक्त एकदाच करायची process आहे.

Installation Steps:

  1. Installer डाउनलोड करा
    खालील "Download Now" बटनावर क्लिक करा. तुमच्या ब्राउझरमध्ये `setup.exe` नावाची फाईल डाउनलोड होईल.
  2. Installer Run करा
    डाउनलोड केलेली फाईल open करा आणि स्क्रीनवरच्या सूचनेनुसार ॲप्लिकेशन install करा.
  3. Protocol Register करा (फक्त एकदाच)
    Install झाल्यावर, app सुरू होईल. त्यात तुम्हाला "Register Protocol" बटनावर क्लिक करायला सांगितले जाईल. ते बटन क्लिक करा आणि permission approve करा. हे खूप गरजेचं आहे, जेणेकरून तुमचा ब्राउझर त्या app सोबत communicate करू शकेल.
  4. काम झालं!
    Setup complete झाल्यावर, तुम्ही हे ॲप्लिकेशन बंद करू शकता. परत मागच्या पेजवर जाऊन "Sign Document" बटनावर पुन्हा क्लिक करा आणि साइन करण्याची process सुरू करा.
