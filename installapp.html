Digital Signer Application - First-Time Setup

To sign documents, a small helper application needs to be installed on your computer. This is a secure, one-time process.

Installation Steps:

  1. Download the Installer
    Click the "Download Now" button below. Your browser will download the `setup.exe` file.
  2. Run the Installer
    Open the downloaded file and follow the on-screen prompts to install the application.
  3. Register the Protocol (One-Time Action)
    After installation, the app will run. It will prompt you to click a "Register Protocol" button. Please click it and approve the administrative prompt. This is essential for the browser to communicate with the app.
  4. You're Done!
    Once setup is complete, you can close the helper application. Return to the previous page and click the "Sign Document" button again to start the signing process.
